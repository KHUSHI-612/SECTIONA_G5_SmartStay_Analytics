# 📖 Data Dictionary: Understanding the Fields

This document explains the key columns found in our processed dataset (`data/processed/tableau_ready.csv`).

| Field Name | Description | Business Relevance |
| :--- | :--- | :--- |
| `id` | Unique identifier for the Airbnb listing. | Data joining. |
| `neighbourhood_cleansed` | The specific Albany Ward (e.g., 6th Ward). | Location-based performance analysis. |
| `property_type` | The type of listing (e.g., Entire Rental Unit, Private Room). | Inventory segmentation. |
| `room_type` | Category of space (Entire home/apt vs. Private room). | Pricing and ROI modeling. |
| `accommodates` | Maximum number of guests. | Capacity planning. |
| `bedrooms` | Number of bedrooms. | Supply-side analysis. |
| `price` | The nightly rate in USD. | Revenue calculation baseline. |
| `host_is_superhost` | Binary (t/f) indicating Superhost status. | Trust and performance correlation. |
| `review_scores_rating` | Overall guest rating (0-5 stars). | Quality benchmark. |
| `estimated_revenue_l365d` | Projected annual revenue based on the SF model. | **Primary ROI Metric.** |
| `estimated_occupancy_l365d` | Projected number of days booked per year. | Asset utilization metric. |
| `revpar` | Revenue Per Available Room. | Industry standard for financial health. |
| `instant_bookable` | Whether guests can book without host approval. | Conversion rate driver. |
